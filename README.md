# Auto React
This bot auto-reacts to specific channels

## Requirements
 * [NodeJs](https://nodejs.org/)

## Setup
```sh
# install dependencies
npm i
# start the bot
npm start
```

## Examples
How to add a channel:
- .add #channelname

How to remove a channel:
- .remove #channelname

How to autoreact the last 100 messages in a channel:
- .react
